Beberapa hal yang harus diperhatikan sebelum integrasi :

- Dapatkan `Secret Key` & `Member Code` di halaman [Pengaturan Secret Key ](https://member.tokovoucher.id/pengaturan/secret-key)
- Silahkan set `Content-Type` pada header menjadi `application/json` jika menggunakan method POST
- Seluruh transaksi via API akan menggunakan method `GET` & `POST`

# ⚠️ Penting !!!

- **Semua HTTP Error** harus diset sebagai transaksi **PENDING** <br />
- **NETWORK TIMEOUT** harus diset sebagai transaksi **PENDING** <br />
- Pastikan menunggu callback final dari tokovoucher untuk mendapatkan status transaksi <br />
- Atau bisa mengecek status transaksi secara berkala setiap 10 menit untuk mendapatkan status transaksi <br />
