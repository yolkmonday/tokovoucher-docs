---
title: 'Transfer Bank'
description: Melakukan transfer uang ke rekening bank
---

API ini digunakan untuk melakukan transfer uang ke rekening bank tujuan.

### Endpoint
<div style="font-family:monospace;margin-bottom: 12px;">
<span style="color:white;background:#27ae60;border-radius: 4px;padding: 4px;">POST</span>
</div>

```
https://api.tokovoucher.net/v1/transfer/bank
```

> Lihat daftar kode bank yang tersedia di [Kode Bank](https://member.tokovoucher.net/list-bank)

### Parameter

| Parameter   | Deskripsi                                                                 | Tipe Data | Wajib |
|-------------|---------------------------------------------------------------------------|-----------|-------|
| ref_id      | ID referensi unik dari Anda (untuk tracking)                              | String    | Ya    |
| bank        | Kode bank tujuan (lihat [Kode Bank](https://member.tokovoucher.net/list-bank)) | String    | Ya    |
| tujuan      | Nomor rekening penerima                                                   | String    | Ya    |
| nominal     | Jumlah uang yang akan ditransfer                                          | Integer   | Ya    |
| member_code | Kode member Anda                                                          | String    | Ya    |
| signature   | Signature dengan formula `md5(MEMBER_CODE:SECRET:REF_ID)`                 | String    | Ya    |

> **Cara membuat signature:** Gabungkan `MEMBER_CODE`, `SECRET`, dan `REF_ID` dengan tanda titik dua (:), lalu hash dengan MD5.
> Contoh: `md5("M001:secretkey123:TRX001")`

### Contoh Request

```json
{
    "member_code": "M2305TEXTXX353GC",
    "ref_id": "TESTFD21",
    "bank": "mandiri",
    "tujuan": "90000000000",
    "signature": "9c0c50f16xxxxfb5dd106f718e3b83dd9",
    "nominal": 12000
}
```

### Response Sukses

```json
{
    "status": "sukses",
    "message": "Transfer berhasil diproses",
    "ref_id": "TESTFD21",
    "trx_id": "T240115ABCD12345",
    "sisa_saldo": 988000
}
```

### Response Gagal

```json
{
    "status": "gagal",
    "message": "Saldo tidak mencukupi"
}
```
