---
title: 'Cek Saldo'
description: Mengecek saldo akun member Anda
---

API ini digunakan untuk mengecek sisa saldo pada akun member Anda.

### Endpoint
<div style="font-family:monospace;margin-bottom: 12px;">
<span style="color:white;background:#e74c3c;border-radius: 4px;padding: 4px;">GET</span>
</div>

```
https://api.tokovoucher.net/member?member_code=[your_member_code]&signature=your_signature_default
```

### Parameter

| Parameter   | Deskripsi                                                                 | Tipe Data | Wajib |
|-------------|---------------------------------------------------------------------------|-----------|-------|
| member_code | Kode member Anda                                                          | String    | Ya    |
| signature   | Signature default Anda (bisa dilihat di [Member Area](https://member.tokovoucher.net/pengaturan/secret-key)) | String    | Ya    |

### Response Sukses

```json
{
    "status": 1,
    "rc": 200,
    "message": "Detail member",
    "data": {
        "saldo": 1000,
        "nama": "Tokovoucher Dev",
        "member_code": "MEMBER20220801UAXH95"
    },
    "ts": 1660204457
}
```

| Field       | Deskripsi                    |
|-------------|------------------------------|
| saldo       | Jumlah saldo Anda saat ini   |
| nama        | Nama akun member             |
| member_code | Kode member Anda             |

### Response Gagal

```json
{
    "status": 0,
    "error_msg": "Member tidak ditemukan !!"
}
```
