Toko Voucher menyediakan bot Telegram yang memudahkan Anda melakukan transaksi dan cek informasi akun langsung dari Telegram.

---

## Cara Menautkan Akun

Ikuti langkah berikut untuk menghubungkan akun member Anda dengan bot Telegram:

1. **Buat grup Telegram baru** - Buat grup khusus untuk bot Toko Voucher
2. **Tambahkan bot** - Invite `@tokovoucherbot` ke dalam grup dan jadikan sebagai admin
3. **Aktifkan bot** - Ketik `/start` di grup. Bot akan menampilkan Group ID
4. **Simpan Group ID** - Copy ID tersebut ke [Form Pengaturan Chatbot](https://member.tokovoucher.net/pengaturan/chatbot)

---

## Perintah yang Tersedia

### Cek Informasi Akun

Untuk melihat informasi akun dan saldo:

```
info
```

---

### Request Tiket Deposit

Untuk membuat tiket deposit, gunakan salah satu format berikut:

**Format 1:**
```
tiket.[KODE_BANK].[NOMINAL]
```

**Format 2:**
```
depo.[KODE_BANK].[NOMINAL]
```

**Contoh:**

Deposit Rp 100.000 via BCA:
```
tiket.bca.100000
```

atau

```
depo.bca.100000
```

> **Catatan:** Untuk daftar kode bank, lihat di [Kode Bank](/kode-bank)

---

## Tips Penggunaan

- Pastikan bot sudah menjadi **admin** di grup
- Satu grup hanya bisa ditautkan ke **satu akun member**
- Semua notifikasi transaksi akan dikirim ke grup yang sudah ditautkan
