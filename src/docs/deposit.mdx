title: 'Deposit'

---


Untuk menerbitkan tiket deposit 

### Endpoint
<div style="font-family:monospace;margin-bottom: 12px;">
<span style="color:white;background:#e74c3c;border-radius: 4px;padding: 4px;">GET</span>
</div>

```
https://api.tokovoucher.id/v1/deposit?member_code=[YOUR_MEMBER_CODE]&secret=[YOUR_SECRET_KEY]&nominal=[NOMINAL]&kode=[KODE_BAYAR]
```

### Deskripsi
Berikut ini parameter / payload yang di harapkan :


| Parameter   | Deskripsi                                                                                                              | Tipe Data | Wajib |
|-------------|------------------------------------------------------------------------------------------------------------------------|-----------|-------|
| member_code | Member code anda                                                                                                       | String    | Ya    |
| secret   | Secret Key, bisa dilihat pada member area  [Lihat Disini](https://member.tokovoucher.id/pengaturan/secret-key)| String    | Ya    |
| nominal | Nominal Deposit yang di inginkan | Int | Ya |
| kode | Kode dari Metode Pembayaran yang di gunakan [Lihat Disini](/kode-bank) | String | Ya|
