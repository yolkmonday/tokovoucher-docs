---
title: 'E-Wallet Reload'
description: Melakukan top up saldo e-wallet
---

API ini digunakan untuk melakukan top up (reload) ke berbagai e-wallet seperti OVO, GoPay, Dana, dll.

### Endpoint
<div style="font-family:monospace;margin-bottom: 12px;">
<span style="color:white;background:#27ae60;border-radius: 4px;padding: 4px;">POST</span>
</div>

```
https://api.tokovoucher.net/v1/transfer/ewallet
```

> Lihat daftar kode e-wallet yang tersedia di [Kode E-Wallet](https://member.tokovoucher.net/list-wallet)

### Parameter

| Parameter   | Deskripsi                                                                 | Tipe Data | Wajib |
|-------------|---------------------------------------------------------------------------|-----------|-------|
| ref_id      | ID referensi unik dari Anda (untuk tracking)                              | String    | Ya    |
| ewallet     | Kode e-wallet tujuan (lihat [Kode E-Wallet](https://member.tokovoucher.net/list-wallet)) | String    | Ya    |
| tujuan      | Nomor e-wallet tujuan                                                     | String    | Ya    |
| nominal     | Jumlah yang akan ditransfer                                               | Integer   | Ya    |
| member_code | Kode member Anda                                                          | String    | Ya    |
| signature   | Signature dengan formula `md5(MEMBER_CODE:SECRET:REF_ID)`                 | String    | Ya    |

> **Cara membuat signature:** Gabungkan `MEMBER_CODE`, `SECRET`, dan `REF_ID` dengan tanda titik dua (:), lalu hash dengan MD5.
> Contoh: `md5("M001:secretkey123:TRX001")`

### Contoh Request

```json
{
    "member_code": "M2305TEXTXX353GC",
    "ref_id": "TESTFD21",
    "ewallet": "OVO",
    "tujuan": "081234567890",
    "signature": "9c0c50f16xxxxfb5dd106f718e3b83dd9",
    "nominal": 50000
}
```

### Response Sukses

```json
{
    "status": "sukses",
    "message": "Top up berhasil diproses",
    "ref_id": "TESTFD21",
    "trx_id": "T240115EFGH67890",
    "sisa_saldo": 950000
}
```

### Response Gagal

```json
{
    "status": "gagal",
    "message": "Nomor e-wallet tidak valid"
}
```
