## Pencarian Produk

Pencarian produk bisa menggunakan Kode Produk Spesifik atau juga prefix nya.
> Misalnya: <br/>
> FF untuk prefix Free Fire akan menampilkan Produk dengan awal FF <br/>
> FF5 akan menampilkan produk spesifik

### Endpoint
<div style="font-family:monospace;margin-bottom:12px;">
<span style="color:white;background:#e74c3c;border-radius: 4px;padding: 4px;">GET</span>
</div>

```
https://api.tokovoucher.net/produk/code?member_code=[member_code]&signature=[signature]&kode=[kode]
```

### Deskripsi
Berikut ini parameter / payload yang di harapkan :


| Parameter   | Deskripsi                                                                                                              | Tipe Data | Wajib |
|-------------|------------------------------------------------------------------------------------------------------------------------|-----------|-------|
| member_code | Member code anda                                                                                                       | String    | Ya    |
| kode | Bisa menggunakan prefix atau kode produk spesifik                                                                                                    | String    | Ya    |
| signature   | Signature default anda Note: Signature detault bisa dilihat pada member area | String    | Ya    |

### Sampel Response Gagal IP Not Allowed

```json
{
    "status":0,
    "error_msg":"Ip Not Allow"
}
```

### Sampel Response Gagal Signature Invalid

```json
{
    "status":0,
    "error_msg":"Signature Invalid"
}
```

### Sampel Response Produk Tidak Ditemukan

```json
{
    "status": 0,
    "rc": 500,
    "error_msg": "Kode Produk Tidak Ditemukan"
}
```
