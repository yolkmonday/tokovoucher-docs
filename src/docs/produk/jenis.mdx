---
title: 'List Jenis Produk'
description: Melihat daftar jenis produk berdasarkan operator
---

API ini digunakan untuk mendapatkan daftar jenis produk berdasarkan operator tertentu.

### Endpoint
<div style="font-family:monospace;margin-bottom:12px;">
<span style="color:white;background:#e74c3c;border-radius: 4px;padding: 4px;">GET</span>
</div>

```
https://api.tokovoucher.net/member/produk/jenis/list?member_code=[MEMBER_CODE]&signature=[SIGNATURE]&id=[id_operator]
```

### Parameter

| Parameter   | Deskripsi                                                                 | Tipe Data | Wajib |
|-------------|---------------------------------------------------------------------------|-----------|-------|
| member_code | Kode member Anda                                                          | String    | Ya    |
| id          | ID operator produk (dari API List Operator)                               | String    | Ya    |
| signature   | Signature default Anda (lihat di [Member Area](https://member.tokovoucher.net/pengaturan/secret-key)) | String    | Ya    |

---

## Response

### Sukses

```json
{
    "status": 1,
    "rc": 200,
    "message": "Data Found",
    "data": [
        {
            "id": 1,
            "operator_nama": "Free Fire",
            "nama": "Otomatis",
            "format_form": "1",
            "status": 1
        },
        {
            "id": 2,
            "operator_nama": "Free Fire",
            "nama": "Via Login",
            "format_form": "2",
            "status": 1
        }
    ],
    "ts": 1661498101
}
```

| Field         | Deskripsi                                                |
|---------------|----------------------------------------------------------|
| id            | ID jenis produk                                          |
| operator_nama | Nama operator                                            |
| nama          | Nama jenis produk                                        |
| format_form   | Format input form (1=standar, 2=dengan server, 3=lainnya)|
| status        | Status aktif (1 = aktif)                                 |

### IP Tidak Diizinkan

```json
{
    "status": 0,
    "error_msg": "Ip Not Allow"
}
```

### Signature Tidak Valid

```json
{
    "status": 0,
    "error_msg": "Signature Invalid"
}
```
