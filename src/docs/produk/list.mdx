---
title: 'List Produk'
description: Melihat daftar produk berdasarkan jenis
---

API ini digunakan untuk mendapatkan daftar produk berdasarkan jenis produk tertentu.

### Endpoint
<div style="font-family:monospace;margin-bottom:12px;">
<span style="color:white;background:#e74c3c;border-radius: 4px;padding: 4px;">GET</span>
</div>

```
https://api.tokovoucher.net/member/produk/list?member_code=[MEMBER_CODE]&signature=[SIGNATURE]&id_jenis=[id_jenis_produk]
```

### Parameter

| Parameter   | Deskripsi                                                                 | Tipe Data | Wajib |
|-------------|---------------------------------------------------------------------------|-----------|-------|
| member_code | Kode member Anda                                                          | String    | Ya    |
| id_jenis    | ID jenis produk (dari API List Jenis)                                     | String    | Ya    |
| signature   | Signature default Anda (lihat di [Member Area](https://member.tokovoucher.net/pengaturan/secret-key)) | String    | Ya    |

---

## Response

### Sukses

```json
{
    "status": 1,
    "rc": 200,
    "message": "Data Found",
    "data": [
        {
            "id": 1,
            "code": "ML5",
            "category_name": "Voucher Game",
            "operator_produk": "Free Fire",
            "jenis_name": "Otomatis",
            "nama_produk": "Mobile Legend 5 DM",
            "deskripsi": "Top up 5 diamond Mobile Legends",
            "price": 1000,
            "status": 1
        },
        {
            "id": 2,
            "code": "ML10",
            "category_name": "Voucher Game",
            "operator_produk": "Free Fire",
            "jenis_name": "Otomatis",
            "nama_produk": "Mobile Legend 10 DM",
            "deskripsi": "Top up 10 diamond Mobile Legends",

```

| Field          | Deskripsi                     |
|----------------|-------------------------------|
| id             | ID produk                     |
| code           | Kode produk (untuk transaksi) |
| category_name  | Nama kategori                 |
| operator_produk| Nama operator                 |
| jenis_name     | Nama jenis produk             |
| nama_produk    | Nama produk                   |
| deskripsi      | Deskripsi produk              |
| price          | Harga produk                  |
| status         | Status aktif (1 = aktif)      |

### IP Tidak Diizinkan

```json

    "status": 0,
    "error_msg": "Ip Not Allow"
}
```

### Signature Tidak Valid

```json
{
    "status": 0,
```
