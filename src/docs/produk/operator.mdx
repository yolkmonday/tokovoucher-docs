---
title: 'List Operator Produk'
description: Melihat daftar operator berdasarkan kategori
---

API ini digunakan untuk mendapatkan daftar operator produk berdasarkan kategori tertentu.

### Endpoint
<div style="font-family:monospace;margin-bottom:12px;">
<span style="color:white;background:#e74c3c;border-radius: 4px;padding: 4px;">GET</span>
</div>

```
https://api.tokovoucher.net/member/produk/operator/list?member_code=[MEMBER_CODE]&signature=[SIGNATURE]&id=[id_category_produk]
```

### Parameter

| Parameter          | Deskripsi                                                                 | Tipe Data | Wajib |
|--------------------|---------------------------------------------------------------------------|-----------|-------|
| member_code        | Kode member Anda                                                          | String    | Ya    |
| id                 | ID kategori produk (dari API List Kategori)                               | String    | Ya    |
| signature          | Signature default Anda (lihat di [Member Area](https://member.tokovoucher.net/pengaturan/secret-key)) | String    | Ya    |

---

## Response

### Sukses

```json
{
    "status": 1,
    "rc": 200,
    "message": "Data Found",
    "data": [
        {
            "id": 1,
            "nama_category": "Voucher Game",
            "nama": "Free Fire",
            "min_karakter": 10,
            "max_karakter": 15,
            "logo": "-",
            "status": 1
        }
    ],
    "ts": 1661497796
}
```

| Field         | Deskripsi                              |
|---------------|----------------------------------------|
| id            | ID operator                            |
| nama_category | Nama kategori                          |
| nama          | Nama operator                          |
| min_karakter  | Minimal karakter untuk nomor tujuan    |
| max_karakter  | Maksimal karakter untuk nomor tujuan   |
| logo          | URL logo operator                      |
| status        | Status aktif (1 = aktif)               |

### IP Tidak Diizinkan

```json
{
    "status": 0,
    "error_msg": "Ip Not Allow"
}
```

### Signature Tidak Valid

```json
{
    "status": 0,
    "error_msg": "Signature Invalid"
}
```
