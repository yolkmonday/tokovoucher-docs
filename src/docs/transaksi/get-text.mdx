---

API ini digunakan untuk melakukan pembelian produk melalui software seperti Otomax, IRS, dan software pulsa lainnya. Response berupa teks (bukan JSON).

### Endpoint
<div style="font-family:monospace;margin-bottom:12px;">
<span style="color:white;background:#e74c3c;border-radius: 4px;padding: 4px;">GET</span>
</div>

```
http://trx-ip.tokovoucher.net/trx?ref_id=[ref_id]&produk=[kode_produk]&tujuan=[tujuan]&secret=[secret]&member_code=[member_code]&server_id=[server_id]
```

### Parameter

| Parameter   | Deskripsi                                                                 | Tipe Data | Wajib |
|-------------|---------------------------------------------------------------------------|-----------|-------|
| ref_id      | ID referensi dari software yang Anda gunakan                              | String    | Ya    |
| produk      | Kode produk di Toko Voucher                                               | String    | Ya    |
| tujuan      | Nomor tujuan (Player ID, User ID, atau Nomor HP)                          | String    | Ya    |
| server_id   | Server ID game (kosongkan jika tidak diperlukan)                          | String    | Tidak |
| member_code | Kode member Anda                                                          | String    | Ya    |
| secret      | Secret Key Anda (lihat di [Member Area](https://member.tokovoucher.net/pengaturan/secret-key)) | String    | Ya    |

> **Tips:** Parameter `tujuan` bisa digabung dengan `server_id` jika diperlukan.
