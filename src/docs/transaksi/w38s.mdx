### Deskripsi
Berikut ini penjelasan form :
> Untuk tujuan bisa menggabungkan tujuan + server_id

| Form   | Deskripsi                                                                                                              | Tipe Data | Wajib |
|-------------|------------------------------------------------------------------------------------------------------------------------|-----------|-------|
| Endpoint      | URL Tokovoucher, Isi dengan <br/> **http://trx-ip.tokovoucher.id/w38s**   | String    | Ya    |
| Member Code   | Member code anda bisa dilihat pada member area [Pengaturan Secret Key ](https://member.tokovoucher.id/pengaturan/secret-key)                                                                                                       | String    | Ya    |
| Signature   | Signature Default Anda Note: Signature bisa dilihat pada member area [Pengaturan Secret Key ](https://member.tokovoucher.id/pengaturan/secret-key) | String    | Ya    |
