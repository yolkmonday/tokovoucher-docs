Melakukan pembelian produk menggunakan Metode GET dan response text untuk software W38S

### Endpoint

```
http://trx-ip.tokovoucher.net/w38s
```

### Panduan

![alt text](/w38s.png)

### Deskripsi
Berikut ini penjelasan form :
> Untuk tujuan bisa menggabungkan tujuan + server_id

| Form   | Deskripsi                                                                                                             | Wajib |
|-------------|-----------------------------------------------------------------------------------------------------------------------|-------|
| Jenis Server      | Pilih Otomax  | Ya    |
| URL Transaksi      | URL Tokovoucher, Isi dengan <br/> **http://trx-ip.tokovoucher.net/w38s**      | Ya    |
| User ID   | Member code anda bisa dilihat pada member area [Pengaturan Secret Key ](https://member.tokovoucher.net/pengaturan/secret-key)                                                                                                          | Ya    |
| PIN   | Isi dengan 0000   | Ya    |
| Password   | Isi dengan Signature Default Anda, Note: Signature bisa dilihat pada member area [Pengaturan Secret Key ](https://member.tokovoucher.net/pengaturan/secret-key)    | Ya    |
