API ini digunakan untuk melakukan pembelian produk melalui software W38S.

### Endpoint

```
http://trx-ip.tokovoucher.net/w38s
```

---

## Panduan Konfigurasi

![Panduan W38S](/w38s.png)

### Pengisian Form

| Form          | Isi Dengan                                                                           | Wajib |
|---------------|--------------------------------------------------------------------------------------|-------|
| Jenis Server  | Pilih **Otomax**                                                                     | Ya    |
| URL Transaksi | `http://trx-ip.tokovoucher.net/w38s`                                                 | Ya    |
| User ID       | Kode member Anda (lihat di [Member Area](https://member.tokovoucher.net/pengaturan/secret-key)) | Ya    |
| PIN           | Isi dengan `0000`                                                                    | Ya    |
| Password      | Signature Default Anda (lihat di [Member Area](https://member.tokovoucher.net/pengaturan/secret-key)) | Ya    |

> **Tips:** Parameter tujuan bisa digabung dengan server_id jika produk membutuhkan.
