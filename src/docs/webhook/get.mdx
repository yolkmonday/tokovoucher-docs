> Kami akan mengirimkan report kepada anda setiap ada perubahan status transaksi
> Pastikan untuk Whitelist IP Kami : 188.166.243.56
Tokovoucher akan mengirimkan balasan SUKSES / GAGAL ke url Anda dengan format sebagai berikut:

### Deskripsi
| Parameter   | Deskripsi                                                                                                              |
|-------------|------------------------------------------------------------------------------------------------------------------------|
| serverid      | Id transaksi / ref id Toko Voucher                                                                                                   | 
| clientid      |Id transaksi / ref id client                                                                                                       | 
| statuscode      | Status transaksi server 1=sukses 2=gagal                                     | 
| kp   | Kode produk client                                                            | 
| msisdn | Nomor tujuan transaksi                                                                                                     | 
| sn   | Nomor seri transaksi / SN operator| 
| msg   | Keterangan / pesan transaksi| 



### Sampel Report Sukses

```
https://url-mitra.com/report?serverid=T220907KLNU024755&clientid=1541622&statuscode=1&kp=SP10&msisdn=081311223344&sn=02806600001851150526&msg=TRXID:T220907KLNU024755. REFID:1541622. SUKSES, SN:02806600001851150526. HRG:10145. SALDO:126700. KP:SP10.081311223344 TGL:2022-06-21
```
