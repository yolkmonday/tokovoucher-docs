---
title: 'Webhook POST / GET'
description: Melakukan pengecekan status transaksi
---

Ini merupakan sampel webhook yang kami kirimkan kepada anda jika menggunakan transaksi melalui metode POST / GET

> Kami akan mengirimkan report ke webhook anda setiap ada perubahan status transaksi
> Pastikan untuk Whitelist IP Kami : 188.166.243.56

### Header
Header yang kami kirimkan dengan Key `X-TokoVoucher-Authorization` dan value dengan Formula `md5(MEMBER_CODE:SECRET:REF_ID)` :

```
"X-TokoVoucher-Authorization": "bd134207f74532a8b094676c4a2ca9ed" //md5(MEMBER_CODE:SECRET:REF_ID)
```

### Sampel Webhook Sukses

```json
{
  "status": "sukses",
  "message": "asd",
  "sn": "ads",
  "ref_id": "tes12",
  "trx_id": "123jad",
  "produk": "FF5",
  "sisa_saldo": 1000000,
  "price": 10000
}
```
